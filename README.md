# Contributing to Datapacks

### 1. Get added to the repo - include your GitHub account in the signup or message @Asometric#2711.

### 2. Clone the repository.
   For easy testing, you can clone it inside of a test world and name the folder datapacks.

![image](https://user-images.githubusercontent.com/40923191/185394043-af9c24ab-32a5-4660-a2e7-5f87f714f9cf.png)


### 3. Make a new branch and name it after your team.

![newbranch](https://user-images.githubusercontent.com/40923191/181110934-99bef5d5-238c-42d9-8725-d6c6a2199f71.png)


### 4. Create your datapack in the repo.

![image](https://user-images.githubusercontent.com/40923191/185395001-ac46dc00-5917-4e89-ae0e-f187cdc7b480.png)


### 5. Every now and then, commit and push your changes to your branch.
